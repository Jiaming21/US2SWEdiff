## Training

Data root: `./datasets/laplacian-swe/`.
`train/` is automatically generated from `train_A` + `train_B` via `make_dataset_aligned.py`

One-shot command:

  bash scripts/train_breastca_256p.sh laplacian

Equivalent `train.py` command (same as script; W&B enabled by default, remove `--use_wandb` and `--wandb_project_name` if not needed):

  python train.py \
    --dataroot ./datasets/laplacian-swe \
    --name laplacian2swe_pix2pix_256 \
    --model pix2pix \
    --dataset_mode aligned \
    --direction AtoB \
    --netG unet_256 \
    --norm batch \
    --pool_size 0 \
    --lambda_L1 100 \
    --batch_size 4 \
    --load_size 256 \
    --crop_size 256 \
    --preprocess resize_and_crop \
    --n_epochs 100 \
    --n_epochs_decay 100 \
    --use_wandb \
    --wandb_project_name GAN

Checkpoint output: `./checkpoints/laplacian2swe_pix2pix_256/`

---

## Testing

`test/` is generated from `test_A` + `test_B` via `make_dataset_aligned.py`

One-shot command:

  bash scripts/test_breastca_256p.sh laplacian 10

The second argument is epoch; `latest` means the latest checkpoint.

Equivalent `test.py` command:

  python test.py \
    --dataroot ./datasets/laplacian-swe \
    --name laplacian2swe_pix2pix_256 \
    --epoch 10 \
    --model pix2pix \
    --dataset_mode aligned \
    --direction AtoB \
    --netG unet_256 \
    --norm batch \
    --load_size 256 \
    --crop_size 256 \
    --preprocess resize_and_crop \
    --num_test 100000

Default result directory (when `--results_dir` is not specified): `./results/laplacian2swe_pix2pix_256/test_10/images/`
