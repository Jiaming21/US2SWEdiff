# DBIM (Diffusion Bridge Implicit Models)

Main set is **Lap|SWE**; three infer sets are **Lap|Lap** (left and right are the **same** Laplacian image, as a target-domain placeholder).
Dataset: generated from the same source data as pix2pix
Dataset in this repository: /n/holylfs05/LABS/zhuang_lab/Lab/Jiaming/US2SWEdiff/DBIM/assets/datasets/breastca_laplacian2swe
Working directory: cd /n/holylfs05/LABS/zhuang_lab/Lab/Jiaming/US2SWEdiff/DBIM

---------------------------------------------------------------------------------
## 0) Data directories

- Main set output: `assets/datasets/breastca_laplacian2swe/`
  - `train/`: BreastCA **train** (real Lap|SWE)
  - `val/`: **BreastCA official test** (both DBIM `EdgesDataset` eval and `sample.py` test split read this directory)
  - `test/`: copy with **same content** as `val/` (name aligned with BreastCA "test" only; upstream code does not read `test/`)
- Three infer sets: `assets/datasets/breastca_infer_{BLUSG,BUSBRA,BUSI}/` (one copy each for `train/` and `val/`; **Lap|Lap** wide images).
Generated by `bash assets/datasets/prepare_breastca_laplacian2swe.sh`
from `PIXLAP/infer_<NAME>/test`. If you previously cropped to single narrow images,
you can restore with: `python3 assets/datasets/expand_infer_to_laplap.py`.

## 2) Test

DBIM calls `sample.py` via `scripts/sample.sh`, **not** ABridge `sample_to_eval`. Results are in **`workdir/<weight_filename_without_.pt>/sample_<step>/split=test/...`** (`sample.py` parses step from weight filename, so use checkpoints like `model_000020000.pt`).

Training output directory is usually **`workdir/breastca_l2s-ddbm/`** (consistent with `--exp=breastca_l2s-ddbm` in `train_bridge.sh`).

```bash
conda activate controlnet_new   # or your dependency environment
cd /n/holylfs05/LABS/zhuang_lab/Lab/Jiaming/US2SWEdiff/DBIM
export PYTHONPATH="${PYTHONPATH}:./"

# Point to your trained weights (replace example step with actual filename)
export MODEL_PATH=/n/holylfs05/LABS/zhuang_lab/Lab/Jiaming/US2SWEdiff/DBIM/workdir/breastca_l2s-ddbm/model_001000.pt

# Args: dataset_name  NFE  sampler  [extra]
# For dbim, the 4th argument is eta (0~1); heun has no 4th argument; for dbim_high_order the 4th argument is order 2 or 3
bash scripts/sample.sh breastca_l2s 10 dbim 0.0
```

`sample.sh` also sets `nproc_per_node` based on visible GPU count; when needed use `NPROC_PER_NODE=1 bash scripts/sample.sh ...`.
