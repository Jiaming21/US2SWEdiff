## 1) 配置 (configs/Template-BBDM.yaml 生成 configs/BreastCA_laplacian2swe_256.yaml)

yaml文件里面写了文件路径
训练4000步，每1000步保存一次权重，保存在 results/.../checkpoint/ 下面

---------------------------------------------------------------------------------

## 2) Train

conda activate controlnet_new
cd /n/holylfs05/LABS/zhuang_lab/Lab/Jiaming/US2SWEdiff/BBDM
python3 main.py \
  --config configs/BreastCA_laplacian2swe_256.yaml \
  --train \
  --sample_at_start \
  --save_top \
  --gpu_ids 0 \
  --max_steps 4000 \
  --save_step_interval 1000

### 续训

conda activate controlnet_new
cd /n/holylfs05/LABS/zhuang_lab/Lab/Jiaming/US2SWEdiff/BBDM
CKPT=/n/holylfs05/LABS/zhuang_lab/Lab/Jiaming/US2SWEdiff/BBDM/results/BreastCA_laplacian2swe/BrownianBridge/checkpoint
python3 main.py \
  --config configs/BreastCA_laplacian2swe_256.yaml \
  --train \
  --sample_at_start \
  --save_top \
  --gpu_ids 0 \
  --resume_model "$CKPT/step_model_4000.pth" \
  --resume_optim "$CKPT/step_optim_sche_4000.pth" \
  --max_steps 4000 \
  --save_step_interval 1000
